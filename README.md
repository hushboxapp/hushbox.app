# Vaultick - Secure for now. Saved for later.

Vaultick is a zero-knowledge vault for your digital legacy, built with a focus on privacy, security, and long-term data preservation.

This repository contains the source code for the "Coming Soon" landing page hosted at [vaultick.com](https://vaultick.com).

## Deployment
This site is automatically deployed via GitHub Pages when changes are pushed to the `main` branch.
